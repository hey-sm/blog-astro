# 安装与配置

本章介绍如何安装 TypeScript、配置开发环境，以及创建 TypeScript 项目。

## 安装 TypeScript

### 全局安装

```bash
# 使用 npm
npm install -g typescript

# 使用 yarn
yarn global add typescript

# 使用 pnpm
pnpm add -g typescript

# 验证安装
tsc --version
```

### 项目本地安装（推荐）

```bash
# 初始化项目
npm init -y

# 安装 TypeScript
npm install --save-dev typescript

# 验证
npx tsc --version
```

## 创建 React + TypeScript 项目

### 使用 Create React App

```bash
npx create-react-app my-app --template typescript

cd my-app
npm start
```

### 手动配置现有项目

```bash
# 安装依赖
npm install --save-dev typescript @types/react @types/react-dom

# 创建 tsconfig.json

```

## TypeScript 编译器（tsc）

### 基本命令

```bash
# 编译单个文件
tsc app.ts

# 编译多个文件
tsc app.ts utils.ts

# 使用配置文件编译
tsc

# 监听模式
tsc --watch

# 指定输出目录
tsc --outDir dist

# 生成声明文件
tsc --declaration
```

### 编译选项

```bash
# 指定目标版本
tsc --target ES2020

# 指定模块系统
tsc --module ESNext

# 严格模式
tsc --strict

# 不生成输出文件（仅检查）
tsc --noEmit
```

## 基础 tsconfig.json

### 最小配置

```json filename="tsconfig.json"
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "ESNext",
    "jsx": "react-jsx",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true
  },
  "include": ["src"],
  "exclude": ["node_modules"]
}
```

### React 项目配置

```json filename="tsconfig.json"
{
  "compilerOptions": {
    // 语言和环境
    "target": "ES2020",
    "lib": ["ES2020", "DOM", "DOM.Iterable"],
    "jsx": "react-jsx",
    
    // 模块
    "module": "ESNext",
    "moduleResolution": "bundler",
    "resolveJsonModule": true,
    
    // 输出
    "outDir": "./dist",
    "noEmit": true,
    
    // 类型检查
    "strict": true,
    "noUnusedLocals": true,
    "noUnusedParameters": true,
    "noFallthroughCasesInSwitch": true,
    
    // 互操作性
    "esModuleInterop": true,
    "allowSyntheticDefaultImports": true,
    "forceConsistentCasingInFileNames": true,
    
    // 其他
    "skipLibCheck": true,
    "isolatedModules": true
  },
  "include": ["src"],
  "exclude": ["node_modules", "dist"]

```

```json filename="tsconfig.json"

  "compilerOptions": {
```

## 编辑器配置

### VS Code

**安装扩展**：

- **TypeScript and JavaScript Language Features**（内置）
- **ESLint**
- **Prettier**
- **Error Lens**（可选）

**配置 settings.json**：

```json filename=".vscode/settings.json"
{
  "typescript.tsdk": "node_modules/typescript/lib",
  "typescript.enablePromptUseWorkspaceTsdk": true,
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  },
  "typescript.preferences.importModuleSpecifier": "relative",
  "typescript.updateImportsOnFileMove.enabled": "always"
}
```

### WebStorm / IntelliJ IDEA

TypeScript 支持开箱即用，无需额外配置。

**配置路径**：
- Settings → Languages & Frameworks → TypeScript
- 启用 TypeScript Language Service
- 配置 tsconfig.json 路径

## 开发工具

### ts-node

直接运行 TypeScript 文件：

```bash
# 安装
npm install --save-dev ts-node

# 运行
npx ts-node src/script.ts

# 配置 package.json
{
  "scripts": {
    "dev": "ts-node src/index.ts"
  }
}
```

### tsx（推荐）

更快的 TypeScript 执行器：

```bash
# 安装
npm install --save-dev tsx

# 运行
npx tsx src/script.ts

# 监听模式
npx tsx watch src/script.ts
```

### tsc-watch

监听模式增强：

```bash
# 安装
npm install --save-dev tsc-watch

# 配置 package.json
{
  "scripts": {
    "dev": "tsc-watch --onSuccess \"node dist/index.js\""
  }
}
```

## 常见问题

### 1. 找不到模块

```ts
// ❌ 错误
import React from 'react'
// Cannot find module 'react' or its corresponding type declarations

// ✅ 解决：安装类型定义
npm install --save-dev @types/react
```

### 2. JSX 语法错误

```json filename="tsconfig.json"
{
  "compilerOptions": {
    "jsx": "react-jsx"  // 或 "react" (旧版)
  }
}
```

### 3. 模块解析错误

```json filename="tsconfig.json"
{
  "compilerOptions": {
    "moduleResolution": "bundler",  // Vite
    // 或
    "moduleResolution": "node"      // Webpack
  }
}
```

### 4. 路径别名

```json filename="tsconfig.json"
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["src/*"],
      "@components/*": ["src/components/*"],
      "@utils/*": ["src/utils/*"]
    }
  }
}
```

```ts
// 使用别名
import Button from '@components/Button'
import { formatDate } from '@utils/format'
```

## 最佳实践

### ✅ 使用严格模式

```json filename="tsconfig.json"
{
  "compilerOptions": {
    "strict": true
  }
}
```

### ✅ 启用增量编译

```json filename="tsconfig.json"
{
  "compilerOptions": {
    "incremental": true,
    "tsBuildInfoFile": ".tsbuildinfo"
  }
}
```

### ✅ 配置路径映射

```json filename="tsconfig.json"
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["src/*"]
    }
  }
}
```

### ✅ 排除不必要的文件

```json filename="tsconfig.json"
{
  "exclude": [
    "node_modules",
    "dist",
    "build",
    "**/*.spec.ts",
    "**/*.test.ts"
  ]
}
```
